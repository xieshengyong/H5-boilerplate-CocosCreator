## 在 VSCode 中集成 ESLint 检查，自动修复错误

1、安装 ESLint 插件，点击「扩展」按钮，搜索 ESLint，然后安装即可。

2、其他配置已预设

## 支持的叠加模式


线性减淡（添加）：

    1、SRC_ALPHA DST_ALPHA 
    
    2、SRC_ALPHA ONE

正片叠底：图片需png8

    DST_COLOR ZERO
